# Deuk Agent Workflow

`deuk-agent-workflow` is a wrapper package for the same CLI published as
`deuk-agent-rule`.

The canonical package remains `deuk-agent-rule` for backward compatibility.

```bash
npm install -g deuk-agent-workflow
deuk-agent-workflow init
```

Existing installs keep working:

```bash
npm install -g deuk-agent-rule
deuk-agent-rule init
```
