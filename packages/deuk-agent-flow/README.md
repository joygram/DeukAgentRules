# Deuk Agent Flow

`deuk-agent-flow` is the main Deuk Agent Flow npm package. It currently wraps
the same CLI implementation published as `deuk-agent-rule`.

`deuk-agent-rule` remains available as a compatibility package during the
transition.

```bash
npm install -g deuk-agent-flow
deuk-agent-flow init
```

Compatibility install:

```bash
npm install -g deuk-agent-rule
deuk-agent-rule init
```
